## Installation

`kubegen` is a code generator. Add the package as dev dependency:

```elixir
def deps do
  [
    {:kubegen, "~> 0.1.0", only: :dev, runtime: false}
  ]
end
```

The docs can be found at <https://hexdocs.pm/kubegen>.
